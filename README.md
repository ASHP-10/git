# Git Learning Repo

Learning Git once and for all.

This repo is a practice REPO for understanding Git so that I can finally stop using AI or GUI for git .

- Reading commit history.
- Multi-line commit messages.
- Merging branches back.
- Resolving merge conflicts during branch merges.
- Rebasing commits to replay work on top of another branch.
- Using stash while moving work between branches.
- Tracking a remote repository.
- Understanding remote refs like `origin/main`.
- Seeing how `HEAD` points to the current branch.
- Exploring `.git/` files like `HEAD`, `config`, `refs`, `objects`, and `logs`.
